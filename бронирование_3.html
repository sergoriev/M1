Процесс бронирования можно отменить на всех этапах. Система возвратит пользователя на титульный экран.

Пользователь заполняет краткий план встречи.

Пользователь заполняет повестку встречи.

Пользователь выбирает тип оповещения, один или оба.

Подтвердить бронирование можно путем нажатия на кнопку "Забронировать".

Выбрать участников можно путем нажатия на кнопку "Добавить". Система откроет окно со списком возможных участников.
